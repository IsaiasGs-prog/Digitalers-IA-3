# Digitalers-IA-3

-Crear un paquete estructurado llamado vehiculos que contenga un archivo de inicialización y dos módulos funcionales: autos.py y operaciones.py

-En el primer módulo, definir una clase Auto con un constructor que reciba e inicialice los atributos de instancia marca y modelo, e incluir un método arrancar que retorne una cadena de texto indicando que el vehículo específico está en funcionamiento.

-En el segundo módulo, importar la clase previamente definida y crea una función probar_auto que reciba una marca y un modelo como parámetros, instancie un objeto de la clase Auto y devuelva el valor.


-Finalmente, crea un archivo principal independiente que importe esta función, la invoque con argumentos de prueba y muestre el resultado por pantalla.
